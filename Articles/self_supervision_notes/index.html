This is by far the excellent paper in the category of self-supervised learning to classify unlabelled data.

My notes while reading this paper:

  • Pre-train your network with both labelled and unlabelled data without using labels.

    • Doing this helps your network learn features without being biased towards labelled data, creating generic representation, learning all the common features in the dataset including the unlabelled data.

  • generate pairwise pseudo-labels for unlabelled data

    • since we have generic representation now, we need to somehow use this representation to cluster the unlabelled data

      pairs of unlabelled images are compared using their representation vectors (obtained by passing unlabelled image through generic representation obtained earlier ) to check similarity between two images. if similar, a pairwise pseudo-label is defined to represent such similar cases.

  • minimize the loss function using both given labels and generated pseudo-labels
